# Commit-message-improver
Commit Message Improver is a simple web-based tool that converts basic or unclear commit messages into professional, structured formats using standard conventions.

A tool to convert simple commit messages into professional format.

## Features
- Improve commit messages
- Copy output
- Simple UI
